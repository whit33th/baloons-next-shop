# Product Detail Components

Эта папка содержит модульные компоненты для страницы деталей продукта.

## Компоненты

### ProductHeader

Отображает заголовок страницы с кнопкой "Back to Catalog" и меткой "Premium Collection".

### ProductGallery

Управляет галереей изображений продукта:

- Главное изображение с навигацией
- Миниатюры для быстрого переключения
- Поддержка клавиатуры (стрелки)
- Индикаторы активного изображения

**Props:**

- `images: string[]` - массив URL изображений
- `productName: string` - имя продукта для alt текста
- `activeImage: number` - индекс текущего изображения
- `onImageChange: (index: number) => void` - callback для смены изображения

### ProductInfo

Основная информация о продукте и форма заказа:

- Название и описание
- Цена и остаток на складе
- Выбор количества
- Кнопка добавления в корзину

**Props:**

- `name: string` - название продукта
- `description: string` - описание
- `price: number` - цена
- `inStock: number` - количество на складе
- `quantity: number` - выбранное количество
- `onQuantityChange: (quantity: number) => void` - callback для изменения количества
- `onAddToCart: (quantity: number) => void` - callback для добавления в корзину

### ProductFeatures

Список особенностей продукта (статический контент).

## Оптимизация

Все компоненты обернуты в `React.memo` для предотвращения ненужных ре-рендеров.
Используются `useCallback` хуки в родительском компоненте для стабильных ссылок на функции.
